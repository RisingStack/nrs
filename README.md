## npm registry switcher

Enables switching between the public and private repositories easily.

### Installation

```
npm install -g nrs
```

### Usage

#### Listing the added repositories

```
nrs ls
```

#### Start using a repository

```
nrs use [alias]
```

#### Adding a repository

```
nrs add [alias] [url]
```

#### Deleting a repository

```
nrs rm [alias]
```
